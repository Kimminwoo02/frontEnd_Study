position relative : 나의 원래 위치(static)를 기준으로 이동 absolute : html상 부모가 아니라 포지션상 부모를 기준으로 이동, 포지션상 부모가 없다면 body 기준으로 위치지정 fixed : 브라우저를 기준으로 위치 이동 - position : top, left, right, bottom, z-index(레이어순서) - absolute, fixed 는 float 과 같이 사용할 수 없다. - relative는 static과 기본성격은 같은데 top, left, right, bottom, z-index 등을 추가로 사용할 수 있다. - sticky : top 속성 추가로 사용
